# Installation

Gama application can be installed via several ways even on the same platform, they include:
- **Building gama directly from it's source code**: Not very useful if you are not
  working on the gama project yourself.
- **Installing via a package manager**: Using a tool which will install gama for you and
  offten will automatically update gama.
- **Installing with prebuilt binaries**: You can also download gama yourself and install it.


## Installing on windows

To install gama on your windows machine, you can download the installer at
[Github releases page](https://github.com/ken-morel/gama/releases/latest)

Use the setup to install gama on your device.

## Installing on linux

To install on linux you will need to build [build gama from source](https://github.com/ken-morel/gama).
