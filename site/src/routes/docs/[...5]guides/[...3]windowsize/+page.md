# Full screen and sized windows

You choose a size for the window when you call `gm_init` function,
but you can resize the window with [`gm_resize`](/reference/gama_8h.html#aa65985ca5b8454ee69087f2243276e77)

And make fullscreen with [`gm_fullscreen`](/reference/gama_8h.html#ac1855c15ae19bfb11c9876b5261b9f75)
